# Youtube-AI-Script-Generator
YouTube Transcript → AI Video Script Generator
(Using Gemini + Webshare Rotating Residential Proxies)

## What this tool does

You enter any topic (e.g. “AI Agents”, “Quantum Computing Explained”, “How to Learn Python Fast”).
It searches YouTube for the top 5 relevant videos.
It fetches the English transcript of each video (using rotating residential proxies so YouTube doesn’t block you).
Gemini (gemini-2.5-flash-lite) reads all transcripts and writes a brand-new, original, engaging 5–10 minute YouTube script ready for you to record.

## Requirements

Python 3.9+
A free or paid Webshare account with Rotating Residential proxies enabled
Google API key with Gemini enabled
YouTube Data API v3 key (free quota is enough)
